À propos de cette application

Ceci est une application web de démonstration développée avec Laravel.

  • Fonctionnalités implantées :
    • authentification avec vérification via un fichier csv (public/assets/csv/login.csv) : Connexion OK
    • Source des pages statiques (public/assets/static_content/about.html) (page à propos statique) : OK
    • présentation d'articles sur la page d'accueil : "liste d'articles de la catégorie ""on n'est pas des pigeons"" (10 max)" : Articles OK
    • Sauvegarde du panier d’articles favoris dans COOKIES : OK
    • Option de présentation : thème et bordure et 3 font policies : OK
    • Recherche d'articles par date min max : OK
    • Source du menu : fichiers CSV : BUG les données sont récupérées correctement depuis un fichier CSV mais ne sont pas accessibles dans la vue Laravel ?
    • API JSON : OK
    • Site web Responsive divers ecrans : Responsive par encore gérer, notamment pour les ecrans plus petits
    • triés par ordre chronologique inverse, et/ou en fonction de la place du mot-clé (dans le titre, l’accroche, ou le contenu) et/ou d’un nombre limité à 10 articles (inutile de lister des centaines d’articles) et/ou en fonction de ce que l’utilisateur a spécifié en option. : OK
    • liste des temps de lecture : OK (mais pas dans le menu)
  • Liens intéressants : Connexion, Accueil, Favoris, sponsor.
  • Mots-clés de recherche : voyager, télévision, Crumble, féerique.
